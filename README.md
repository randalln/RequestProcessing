# Simplified Request Processing Android App

## Architecture
* MVVM
* UI Framework: Jetpack Compose
* Kotlin Coroutines and Flow
* Dependency Injection: Hilt (though I prefer [Metro](https://zacsweers.github.io/metro/latest/) these days)

## Additional Libraries Used
* MockK and Turbine for unit testing
* [slide-to-unlock](https://github.com/RevenueCat/slide-to-unlock)
